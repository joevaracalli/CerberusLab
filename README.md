# Cerberus Lab - Music Label Website

A modern, clean one-page website for Cerberus Lab music label built with React and Tailwind CSS.

## Features

- **Hero Section** - Eye-catching introduction with call-to-action buttons
- **Social Media Links** - Quick access to all social platforms
- **New Releases** - Showcase of latest music releases
- **Artist Spotlight** - Featured artists with their social links
- **Contact Form** - Professional contact section for bookings and demos
- **Responsive Design** - Works perfectly on all devices

## Tech Stack

- React 18
- Tailwind CSS
- Vite
- Modern ES6+ JavaScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```

## Customization

- Update artist information in the `artists` array in `src/App.jsx`
- Modify releases in the `releases` array
- Change colors by updating the Tailwind classes
- Add your own background images in the `public` folder
